Beteckna

This font is available from beteckna.se, it is a geometric sans-serif typeface inspired by Paul Renners popular type, Futura. It was drawn by Johan Mattsson in May 2007. The font is free, licensed under terms of the GNU GPL. This version supports English and a few nordic languages. Special character &#x2708; ( ✈ ) depicts two cats.
